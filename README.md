# 后台管理项目

该项目用于学习，主要使用了 vite、react、typescript 等内容。

## 项目结构

- src
  - api 网络请求
  - assets 资源
  - components 公共组件
  - config 配置文件
  - hook 公共 hook
  - layout 布局
  - router 路由
  - types 公共 ts 定义
  - utils 工具函数
  - views 业务页面

## 配置

- tsconfig 应在 tsconfig.app.json 下配置才有效
- vite.config.ts 用于配置 vite
- .vscode 下的 settings.json 用于配置 vscode
- .prettierrc.cjs 用于配置 prettierrc
